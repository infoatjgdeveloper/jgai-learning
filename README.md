# JGAI Learning

The AI University — learn anything with AI-built paths, real Canvas classroom sync, and free JGAI-certified degree-style programs.

Part of the [JG Developer](https://jgdeveloper.com) platform family.

## Stack
React 19 · Vite · Tailwind v4 · Firebase (auth + data) · Claude API (AI) · Vercel (hosting + edge functions)

## Run locally
1. `npm install`
2. Copy `.env.example` to `.env.local` and set `ANTHROPIC_API_KEY`
3. `npm run dev` (AI/Canvas proxies require `vercel dev` or deployment)

## Deploy
Push to GitHub and import into Vercel. Set `ANTHROPIC_API_KEY` in project env vars.
